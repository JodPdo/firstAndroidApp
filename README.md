# Happy Birthday Card App 🎂

A beginner Android application built with **Kotlin** and **Jetpack Compose**.

This project displays a digital birthday card with a background image, greeting message, and signature text using modern Android UI development with Compose.

---

## 📱 Preview

Features included in the app:

* Background image (`androidparty.png`)
* Centered birthday greeting text
* Signature text aligned to the bottom-right area
* Clean Compose-based layout

## ✨ Features

* `Column()` for vertical text layout
* `Box()` for layering image and text
* `Image()` composable for background image
* `Text()` composable for greeting message
* `Modifier.padding()` for spacing
* `Alignment.End` for signature placement
* Responsive Compose UI structure

---

## 🚀 How to Run

1. Open the project in Android Studio
2. Sync Gradle files
3. Connect an Android device or start an emulator
4. Click **Run**
